# يلا · Yalla — Apprendre le libanais

Une application web de **flashcards** pour apprendre l'arabe levantin (dialecte libanais), avec répétition espacée, audio et suivi de progression. Niveau A1–A2, 384 mots et expressions.

> Le libanais, un peu chaque jour.

## Fonctionnalités

- **384 mots et expressions** en libanais authentique, classés en 22 catégories (essentiels, famille, nourriture, transport, verbes, expressions…)
- **Répétition espacée** : chaque mot revient à intervalles croissants (1, 3, 7, 14, 30 jours) selon ce que tu retiens
- **Audio** en arabe (priorité aux voix levantines de l'appareil)
- **Écriture arabe + phonétique + français** sur chaque carte, avec phrases d'exemple
- **Mes mots** : retrouve tes mots vus, filtrés par niveau, avec barre de progression
- **Dictionnaire** complet avec recherche
- **Série de jours** consécutifs pour rester motivé
- Progression **sauvegardée sur l'appareil**, fonctionne hors ligne une fois ouverte
- **Installable sur iPhone** comme une vraie application

## Installation sur iPhone

1. Ouvrir l'adresse de l'app dans **Safari**
2. Toucher le bouton **Partager** (carré avec une flèche)
3. Choisir **« Sur l'écran d'accueil »**
4. L'icône Yalla apparaît comme une application en plein écran

## Audio en libanais

L'app utilise la synthèse vocale de l'appareil. Pour une prononciation plus proche du libanais, installer des voix arabes :
**Réglages → Accessibilité → Contenu énoncé → Voix → Arabe**

Le vocabulaire est intégralement en libanais ; l'accent dépend des voix installées.

## Technique

Un seul fichier `index.html`, sans dépendance ni serveur. Tout fonctionne côté navigateur (HTML, CSS, JavaScript natif).
